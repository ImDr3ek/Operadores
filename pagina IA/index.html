Enciclopédia de Operadores Web

O guia definitivo para dominar a lógica visual e funcional.

1. Operadores de Estrutura (HTML)

No HTML, os operadores definem relações hierárquicas e comportamentais entre componentes.

Atribuição de Propriedades (=)

Conecta chaves a valores dentro das tags.

<img src="foto.jpg" alt="Descrição">

Operador de Âncora (#)

Cria links internos para IDs específicos na mesma página.

<a href="#secao-contato">Pular para contato</a>

Operador Global (*)

Embora usado no CSS, no HTML ele é implícito em seletores de wildcard para scripts.

2. Operadores de Estilo & Lógica (CSS)

O CSS moderno utiliza operadores matemáticos e combinadores para layouts complexos.

Combinador de Irmãos (~)

Seleciona todos os elementos que são irmãos de um anterior.

p ~ span { color: red; }

Operador de Negação (:not)

Aplica estilos a tudo, exceto ao elemento especificado.

div:not(.ativo) { opacity: 0.5; }

Função Matemática (min/max)

Operadores de comparação para responsividade fluida.

font-size: clamp(1rem, 5vw, 3rem);

3. Operadores de Programação (JavaScript)

A lógica computacional pura que dita as regras do sistema.

Nullish Coalescing (??)

Retorna o lado direito se o esquerdo for nulo ou indefinido.

const nome = usuario ?? "Anônimo";

Spread Operator (...)

Expande elementos de um array ou objeto.

const novoArray = [...antigo, 4, 5];

Strict Equality (===)

Compara valor e tipo, evitando bugs de tipagem.

if (status === 200) { active(); }
